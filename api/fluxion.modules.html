fluxion.modules.llm_modules module

A module to handle interacting with a locally hosted language model (LLM) via REST API. Currently supports querying and chatting with the LLM.

Example

```python from fluxion.modules.llm_query_module import LLMQueryModule

# Initialize the LLMQueryModule llm_module = LLMQueryModule(endpoint=”http://localhost:11434/api/generate”, model=”llama3.2”)

# Query the LLM response = llm_module.query(prompt=”What is the capital of France?”) print(response) ```

```python

from fluxion.modules.llm_query_module import LLMChatModule

# Initialize the LLMChatModule llm_module = LLMChatModule(endpoint=”http://localhost:11434/api/chat”, model=”llama3.2”)

# Chat with the LLM response = llm_module.chat(messages= [{

“role”: “user”, “content”: “Hello!” }, { “role”: “assistant”, “content”: “Hello, how can I help you?” }]

)

print(response) ```
